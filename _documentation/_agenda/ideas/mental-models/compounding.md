# Compounding

Growth that builds on itself. Each cycle's output becomes the next cycle's input. The result is exponential, not linear.

Einstein allegedly called compound interest "the eighth wonder of the world." The math applies to far more than money.

## Where it applies
- **Skills**: each hour of deliberate practice builds on previous hours
- **Knowledge**: each connected idea makes the next one easier to understand
- **Reputation**: trust accumulated over time opens doors unavailable to newcomers
- **Code**: a well-designed codebase compounds — each good abstraction makes future features cheaper

## When to use
- Long-term prioritization: small, consistent actions > occasional large efforts
- Evaluating habits: "if I do this daily for a year, what does it compound into?"
- Product design: what behaviors, if users do them repeatedly, compound into deep value?

## Example
Learning one mental model per week for a year = 52 models. But more importantly, models start cross-linking — each new one enriches the entire network. The 50th model is easier to learn and more useful than the 5th.

## Pitfall
Compounding also applies to bad habits, technical debt, and poor culture. The same engine that grows good things grows bad ones.
