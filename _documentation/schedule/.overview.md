# info/organization — Overview

Operational second brain for the Newhaze organization. Contains all planning, tracking, and knowledge management files used in day-to-day work.

## Folder map

| Folder | Purpose |
|---|---|
| `tasks/` | Atomic work items — the lowest unit of tracked work |
| `projects/` | Initiatives grouping tasks toward a milestone |
| `products/` | Product-level strategy and documentation |
| `daily/` | Daily notes in `YYYY-MM-DD.md` format |
| `weekly/` | Weekly reviews in `YYYY-Www.md` format |
| `planning/` | Quarterly goals and long-range strategy |
| `outputs/` | Dated deliverables and research artifacts |
| `context/` | Decision records and background knowledge |
| `people/` | Team member profiles and collaboration context |
| `ideas/` | Inbox for unscoped concepts |

## Taxonomy

Every task and project must have `product:` frontmatter. Products: `newhaze`, `chulistic`, `rabbittotem`, `second-brain`. The hierarchy is: Product → App → Project → Module → Feature → Task.

## File format

All files are Obsidian vault documents using `[[wiki/path/slug]]` link format. Templates are in `.claude/reference/file-formats.md`.
